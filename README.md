# Приложение

### Старт разработки

-   Установить нужную версию Node `nvm use`. [nvm-sh](https://github.com/nvm-sh/nvm)
-   Установи пакеты `npm ci`
-   Запусти локальную сервер для разработки `npm run dev`
-   Собрать релизный бандл можно командой `npm run build`
